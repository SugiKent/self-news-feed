# 開発ルール

- ファイルの編集をしたら `npm run build` を実行してください
- gatsyby のファイルは `self-news-feed` ディレクトリにあります

# Git

branch は `gh-pages` がベースです。commit や pull はこの branch から行ってください

# ドキュメント

- docs 以下のファイルを参照してください

# コマンド

- Gatsby の静的ファイルビルドは `npm run build` を実行してください
- `npm run build` は最大5分間かかるので実行した際は待機してください

# SugiKent セルフニュースフィード 仕様書

## 概要
このサイトは SugiKent が毎日チェックする情報を自動収集・表示する静的サイトです。

## システム構成
- ホスティング: GitHub Pages
- 更新方式: shell script を cron によって定期実行（数時間に1回）
- コンテンツタイプ: 静的HTMLファイル

## 更新プロセス
1. shell script でが定期的に以下を実行:
   - 情報収集（天気、リリース情報など）
   - GitHub リポジトリへの自動プッシュ
   - HTMLなど静的ファイル生成

## 表示情報
- 日付（最新更新日時）
- 品川区の天気情報
- SugiKent がウォッチしているリポジトリのリリース情報

## 技術要件
- Gatsby を利用する
- GitHub Pages で動作する静的ファイル構成
- GitHub API連携（リポジトリ情報取得用）
- 天気API連携
